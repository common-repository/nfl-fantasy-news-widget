NFL Fantasy News Widget for Sports Blog Content by NFLSoup.com

== Description ==

If your blog covers anything having to do with NFL football, Your readers will find this widget very useful. We're aggregating the top fantasy football sites in the United states.

Install this widget and it will be updated in the future with additional data including live scores, and player rankings. We're also writing updates so you will be able to easily edit the skins.

== Installation ==

1. Unzip nfl-fantasy-news-widget.zip into /nfl-fantasy-news-widget folder.
2. /nfl-fantasy-news-widget folder should contain 3 files (nfl-fantasy-news-widget.php, readme.txt and screenshot-1.png)
3. Upload /nfl-fantasy-news-widget folder to your /wp-content/plugins/ directory
4. Activate the plugin through the 'Plugins' menu in WordPress
5. Add the widget to your sidebar (Navigate to Appearance menu > Widgets and drag the widget to the prefered place on your sidebar)
